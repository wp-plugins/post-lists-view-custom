Customize the list of the post and page and the custom post type.

== Description ==

* Posts lists
* Pages lists
* Custom Posts Type lists
* Media lists
* Comments lists
* Available Widgets lists
* Menus to show on screen
* Menus to show advanced menu properties

These lists to Customization is possible.

== Installation ==

1. Upload the entire post-list-view-custom folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You will find 'Post Lists View Customize' menu in your WordPress admin panel.
