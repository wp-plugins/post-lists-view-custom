== Description ==

Customize the list of the post and page.
custom post type page, too.
You can customize the column display items freely.

== Installation ==

1. Upload the entire post-list-view-custom folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You will find 'Post Lists View Customize' menu in your WordPress admin panel.
