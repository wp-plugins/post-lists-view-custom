Customize the list of the post. view thumbnails and custom fields.

== Description ==

This plug-in is a plug-in that Customize the Post Lists.
Incidentally, you can also display thumbnails and custom fields.
Change the display order.

== Installation ==

1. Upload the full directory into your wp-content/plugins directory
2. Activate the plugin at the plugin administration page
3. Open the plugin configuration page,
   which is located under Options -> Post Lists View Costom
